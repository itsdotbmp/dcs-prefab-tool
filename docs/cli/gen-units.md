# `dcs-sms gen-units`

[← CLI reference index](README.md)

regenerate framework/constants/{units,statics}.lua from dcs-lua-datamine

## Usage

```
dcs-sms gen-units [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--datamine` | string | `""` | path to dcs-lua-datamine repo (default: $DCS_LUA_DATAMINE_PATH or D:/git/dcs-lua-datamine) |
| `--out-dir` | string | `""` | where to write units.lua/statics.lua (default: ./framework relative to cwd) |
