# `dcs-sms uninstall-hook`

[← CLI reference index](README.md)

remove the Lua hook + revert the MissionScripting.lua patch

## Usage

```
dcs-sms uninstall-hook [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--dcs-path` | string | `""` | DCS install path (used to revert the MissionScripting.lua patch) |
| `--saved-games` | string | `""` | override Saved Games path |
