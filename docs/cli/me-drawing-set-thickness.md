# `dcs-sms me drawing set-thickness`

[← CLI reference index](README.md)

change a line / polygon drawing's outline thickness

## Usage

```
dcs-sms me drawing set-thickness [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--name` | string | `""` | drawing name (Line / Polygon only) |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--thickness` | float | `0` | thickness in pixels (positive) |
| `--timeout` | duration | `30s` | wall-clock timeout |
