# `dcs-sms me file new`

[← CLI reference index](README.md)

create a new empty mission in the open Mission Editor

## Usage

```
dcs-sms me file new [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--force` | bool | `false` | discard unsaved changes in the current mission |
| `--map` | string | `""` | theatre name (e.g. Syria, Caucasus) |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--timeout` | duration | `30s` | wall-clock timeout |
