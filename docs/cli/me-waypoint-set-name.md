# `dcs-sms me waypoint set-name`

[← CLI reference index](README.md)

set a waypoint's display name (empty string allowed)

## Usage

```
dcs-sms me waypoint set-name [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--group-id` | int | `0` | group id (mutually exclusive with --group-name) |
| `--group-name` | string | `""` | group name (mutually exclusive with --group-id) |
| `--index` | int | `-1` | waypoint index (0-based; required) |
| `--name` | string | `""` | waypoint display name (empty string is legal) |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--timeout` | duration | `30s` | wall-clock timeout |
