# `dcs-sms me drawing set-pos`

[← CLI reference index](README.md)

move a drawing's anchor to a new north/east coordinate

## Usage

```
dcs-sms me drawing set-pos [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--east` | float | `0` | meters east of theatre origin |
| `--name` | string | `""` | drawing name |
| `--north` | float | `0` | meters north of theatre origin |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--timeout` | duration | `30s` | wall-clock timeout |
