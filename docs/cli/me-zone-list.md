# `dcs-sms me zone list`

[← CLI reference index](README.md)

list all zones in the open mission

## Usage

```
dcs-sms me zone list [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--name` | string | `""` | filter by zone-name substring (case-insensitive) |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--shape` | string | `""` | filter by shape: circle \| quad |
| `--timeout` | duration | `30s` | wall-clock timeout |
