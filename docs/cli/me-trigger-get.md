# `dcs-sms me trigger get`

[← CLI reference index](README.md)

return full data for a trigger by name

## Usage

```
dcs-sms me trigger get [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--name` | string | `""` | trigger name (the comment field) |
| `--pretty` | bool | `false` | indent JSON output |
| `--raw` | bool | `false` | return verbatim trigrules entry (no enrichment) |
| `--saved-games` | string | `""` | override Saved Games path |
| `--timeout` | duration | `30s` | wall-clock timeout |
