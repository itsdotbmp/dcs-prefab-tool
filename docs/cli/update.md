# `dcs-sms update`

[← CLI reference index](README.md)

download the latest dcs-sms.exe from GitHub and replace this binary

## Usage

```
dcs-sms update [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--check` | bool | `false` | report whether an update is available without downloading |
