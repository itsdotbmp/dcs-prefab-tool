# `dcs-sms teardown`

[← CLI reference index](README.md)

remove the ME mod and the hook in one shot

## Usage

```
dcs-sms teardown [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--dcs-path` | string | `""` | override DCS install path |
| `--saved-games` | string | `""` | override Saved Games path |
