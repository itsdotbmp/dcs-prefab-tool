# `dcs-sms setup`

[← CLI reference index](README.md)

update dcs-sms.exe, then install the ME mod and the hook in one shot

## Usage

```
dcs-sms setup [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--dcs-path` | string | `""` | override DCS install path |
| `--no-config-save` | bool | `false` | do not persist paths to config |
| `--saved-games` | string | `""` | override Saved Games path |
| `--skip-update` | bool | `false` | skip the self-update step (used internally after re-exec) |
