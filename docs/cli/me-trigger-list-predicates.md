# `dcs-sms me trigger list-predicates`

[← CLI reference index](README.md)

list available trigger predicates (filter by kind / search)

## Usage

```
dcs-sms me trigger list-predicates [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--kind` | string | `""` | filter: condition\|action\|trigger |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--search` | string | `""` | substring to match against name or alias |
| `--timeout` | duration | `30s` | wall-clock timeout |
