# `dcs-sms me trigger describe-predicate`

[← CLI reference index](README.md)

print the field schema for one trigger predicate

## Usage

```
dcs-sms me trigger describe-predicate [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--name` | string | `""` | predicate canonical name (e.g. c_flag_is_true) or alias (flag-is-true) |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--timeout` | duration | `30s` | wall-clock timeout |
