# `dcs-sms uninstall-me-mod`

[← CLI reference index](README.md)

remove the Mission Editor mod (revert MissionEditor.lua, delete modules)

## Usage

```
dcs-sms uninstall-me-mod [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--dcs-path` | string | `""` | override DCS install path |
