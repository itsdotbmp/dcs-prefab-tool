# `dcs-sms me airbase set-coalition`

[← CLI reference index](README.md)

set an airbase's coalition (red, blue, neutral) and refresh the map display

## Usage

```
dcs-sms me airbase set-coalition [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--coalition` | string | `""` | red, blue, or neutral |
| `--name` | string | `""` | airbase name |
| `--pretty` | bool | `false` | indent JSON output |
| `--saved-games` | string | `""` | override Saved Games path |
| `--timeout` | duration | `30s` | wall-clock timeout |
