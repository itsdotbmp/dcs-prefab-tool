# `dcs-sms install-me-mod`

[← CLI reference index](README.md)

install/update the Mission Editor mod into <DCS install>/MissionEditor/

## Usage

```
dcs-sms install-me-mod [flags]
```

## Flags

| Name | Type | Default | Description |
|---|---|---|---|
| `--dcs-path` | string | `""` | override DCS install path |
| `--no-config-save` | bool | `false` | do not persist --dcs-path to config |
