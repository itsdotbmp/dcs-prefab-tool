# `sms.K.statics` ‚Äî catalog of every static-spawnable DCS type

`sms.K.statics` is a generated catalog of every DCS static-object type that can be spawned via [`sms.static.create`](static.md) or vanilla `coalition.addStaticObject`. Sibling of [`sms.K.units`](units.md); same shape, different spawn API. Each entry is a string constant.

`sms.K.statics` is a sub-table of the unified `sms.K` constants namespace ‚Äî see [`constants.md`](constants.md) for the full `sms.K` reference including all other enum topics.

This page is the canonical reference for `sms.K.statics`. The cross-cutting framework rules every API call follows live in [`AGENTS.md`](../../AGENTS.md). The catalog itself is generated by `dcs-sms gen-units` from dcs-lua-datamine ‚Äî do not hand-edit `framework/constants/statics.lua`.

## Categories

| Namespace | Contains | Example |
|---|---|---|
| `sms.K.statics.fortifications` | Statics-only fortifications (bunkers, walls, towers, sandbags). For *group-spawnable* obstacles like `Cars/Car/Bunker`, see [`sms.K.units.fortifications`](units.md). | ‚Äî |
| `sms.K.statics.cargos` | Slingable crates and containers | `sms.K.statics.cargos.container_20ft ‚Üí "container_20ft"` |
| `sms.K.statics.personnel` | Deck crew, statue-soldiers | `sms.K.statics.personnel.us_carrier_tech ‚Üí "us carrier tech"` |
| `sms.K.statics.heliports` | FARPs, helipads, oil rigs | `sms.K.statics.heliports.FARP ‚Üí "FARP"` |
| `sms.K.statics.warehouses` | Warehouse buildings | ‚Äî |
| `sms.K.statics.airfields` | Grass strips | `sms.K.statics.airfields.GrassAirfield ‚Üí "GrassAirfield"` |
| `sms.K.statics.equipment` | Aircraft deck equipment, jet starters, generators | ‚Äî |
| `sms.K.statics.effects` | Smoke, fire, markers | `sms.K.statics.effects.big_smoke ‚Üí "big_smoke"` |
| `sms.K.statics.animals` | Cows, etc. | `sms.K.statics.animals.Cow ‚Üí "Cow"` |
| `sms.K.statics.airships` | LTA vehicles (balloons) | ‚Äî |
| `sms.K.statics.ground_objects` | Misc | ‚Äî |

Identifier sanitization, asset-pack origin tagging, and `sms.K.statics.origin_of` all work the same way as in [`sms.K.units`](units.md). See that page for details.

## Use at the call site

```lua
sms.static.create({
  name     = "fuel-tank",
  type     = sms.K.statics.fortifications.FARP_Fuel_Depot,
  position = {x = 0, y = 0, z = 0},
  country  = sms.K.countries.USA,
  heading  = 45,
})
```
